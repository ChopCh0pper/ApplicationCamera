#  RxJava 1.x Adapter

An `Adapter` for adapting RxJava 1.x types.

Available types:

 * `Observable<T>`
 * `Single<T>`
 * `Completable`


## Set up

Add dependency to your `build.gradle`

```groovy
repositories {
  maven { url 'https://jitpack.io' }
}
 
compile 'io.fotoapparat.fotoapparat:adapter-rxjava:x.y.z'
```
